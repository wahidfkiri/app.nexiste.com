# Laravel Code Review Skill

## Goal

Analyze and improve Laravel code quality.

## Instructions

* Check naming conventions (variables, functions, classes)
* Verify validation rules
* Detect security issues (SQL injection, XSS, CSRF)
* Optimize queries (avoid N+1, use eager loading)
* Suggest clean architecture improvements

## Output

* List of issues
* Suggested fixes
* Improved code snippet (if needed)
